## Synopsis
`gwst manifest preset ls [--no-prompt]`

## Intent
List preset entries in `gwst.yaml`.

## Notes
- This is the manifest-first replacement for the legacy `gwst preset ls`.
- This command is inventory-only and does not run `gwst apply`.
